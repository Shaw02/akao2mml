	■概要
	ＦＦ８，ＦＦ９の抜き出し曲データを、
	MML2MID（http://platz.jp/~mml2mid/）用のMMLに変換します。
	MS-DOSの標準出力ファンクションで、MMLを出力するので、
	ファイルに保存したい場合は、
	ex.)  a:\>ff8mml ff9_001 >ff9_001.mml
	等と、コマンドラインに打ち込んで下さい。

	■注意
	変換によって得られたMMLファイル、MIDIファイルの掲載は禁止です。

	Windows XP（32bit版）では、動作する事を確認しました。
	Windows vistaでは、『動作しない』事を確認しました。
	元々、MS-DOS時代に書いたプログラムなので、諦めてください。
	私は、『Windowsでのソフト開発は知りません。』
	作ってくれてもｏｋ。


	■ソースファイルについて
	Microsoft Macro Assembler version 6.11でアセンブルできます。
	MASM 6.11も、Windows vistaでは使えないようです。
	PC98をLANにつなげて、vistaで編集、PC98でアセンブルしました･･･。

	てか、数年前に作ったものなので、自分自身、
	細かい所でどんな処理しているのか、覚えてなかったりします。


製作のファイルは【FF8MML  .LZH】です。
