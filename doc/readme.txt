	このアプリケーションは、SQUARE ENIX社（旧SQUARE社）の
	SPC版AKAO及び、PS版AKAOフォーマットを
	MML2MID仕様のMML（Music Macro Language）に逆コンパイルします。
	※一部のデータは、正常に逆コンパイル出来ません。

製作のファイルは【AKAO2MML.LZH】です。
