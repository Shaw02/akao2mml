


本アプリケーションは、i80186(V30)のアセンブリ言語で開発されております。
以下の開発環境にてビルドできる事を確認しています。

　(1) "Microsoft (R) Visual C++ 2008 Express Edition with SP1"	（無料）
　　　→http://www.microsoft.com/japan/msdn/vstudio/express/default.aspx
　(2) "Microsoft (R) Macro Assembler version 6.11a





尚、ビルドには別途、16bit版 リンカーが必要になる場合があります。
以下のサイトからダウンロードして使ってください。
　　　→ftp://ftp.microsoft.com/softlib/mslfiles/lnk563.exe

　(1) "Microsoft (R) Visual C++ 2008 Express Edition with SP1"を使う場合。
　　　・ダウンロード後、exeファイルを実行し、解凍して下さい。
　　　・解凍後の"link.exe"を、"link16.exe"にファイル名を換え、
　　　　Visual C++.netをインストールしたフォルダーの"bin"フォルダに保存して下さい。
　　　　例）"C:\Program Files\Microsoft Visual Studio 9.0\VC\bin"

　(2) "Microsoft (R) Windows vista"を使う場合。
　　　・ダウンロード後、exeファイルを実行し、解凍して下さい。
　　　・Macro Assemblerをインストールしたフォルダーの"bin"フォルダーに保存して下さい。
　　　　どうせ、Windows vistaでは標準のlink.exeは使えないので、上書きして下さい。
